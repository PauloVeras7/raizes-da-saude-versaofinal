# Raízes da Saúde — Sistema Completo

Este projeto é a evolução do site estático original [Raízes da Saúde](https://github.com/PauloVeras7/raizes-da-saude), agora com backend em PHP e integração com banco de dados MySQL. Ele funciona como um sistema de agendamento e gerenciamento de consultas médicas.

## Funcionalidades principais
- Cadastro e login de usuários
- Agendamento de consultas
- Listagem e gerenciamento de dados de pacientes
- Proteção de credenciais sensíveis usando `.gitignore`
- Upload de arquivos (como fotos de perfil)
- Controle de acessos para usuários e administradores
- Relatórios básicos de histórico de agendamentos

## Tecnologias utilizadas
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP para ambiente local

## Função desempenhada
- Desenvolvimento completo do backend em PHP
- Criação e integração com banco de dados MySQL
- Implementação das funcionalidades de CRUD
- Proteção de dados sensíveis e gerenciamento de arquivos
- Estruturação do projeto para permitir manutenção e futuras evoluções
